## 📌 About This Project

This is my **personal portfolio website** — designed and developed from scratch to represent who I am as a developer. The site features a warm **gold & dark color theme**, a professional layout, and a focus on clean UI design.

> ⚠️ **Note:** This portfolio was built **for showcase/display purposes only.** Some interactive features (like the contact form) are intentionally non-functional, as this project is meant to demonstrate frontend design and development skills — not serve as a live production app.

---

## ✨ Features

- 🎨 **Gold-themed UI** — elegant, minimal design with a signature color palette
- 👤 **Hero / Intro Section** — personal introduction with a professional photo
- 🛠️ **Skills & Services Section** — highlights what I bring to the table
- 💼 **Portfolio / Work Section** — displays selected projects
- 📬 **Contact Section** — UI-only contact form *(non-functional — for display purposes)*
- 📱 **Responsive Design** — adapts to all screen sizes using Tailwind CSS

---

## 🚫 Non-Functional Features

The following features are **UI-only** and do not have backend functionality:

| Feature | Status | Reason |
|---|---|---|
| Contact Form | ❌ Not functional | No backend / form handler connected |

This is intentional — the project is a **frontend-only showcase**.

## 🚀 Getting Started

No installation or build tools required. Just open it in your browser.

```bash
# 1. Clone the repository
git clone https://github.com/your-username/gold-portfolio.git

# 2. Navigate into the folder
cd gold-portfolio

# 3. Open index.html in your browser
open index.html
```

Or simply drag and drop `index.html` into any browser window.
